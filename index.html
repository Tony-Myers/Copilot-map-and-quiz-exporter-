Copilot Map and Quiz Exporter

Turn Copilot study outputs into interactive quizzes and mind maps

How to use this tool

Standard workflow
  1. Generate a quiz and/or mind map in your Copilot agent.
  2. Copy the full Copilot response.
  3. Paste it into the box below.
  4. Click Save payload if it does not auto-load.
  5. Open the Quiz Tool or Mind Map Tool.
Quick steps
  1. Copy the full Copilot response.
  2. Paste it into the box below.
  3. The tool will try to extract the JSON automatically.
  4. Open the Quiz Tool or Mind Map Tool.
